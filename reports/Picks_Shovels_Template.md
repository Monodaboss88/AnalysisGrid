## 6. HOW TO RUN THIS ANALYSIS

1. **Pick the Industry** â€” Identify top 5-10 end customers
2. **Map Layer 2** â€” Ask: "Who do they ALL depend on?" Look for shared suppliers
3. **Map Layer 3** â€” Ask: "Who do the Layer 2 companies depend on?"
4. **Score each supplier:**
   - Multi-customer count (higher = more durable)
   - Moat type (monopoly > oligopoly > competitive)
   - Gross margin (>40% = pricing power)
   - Debt/Equity (<0.5 = healthy)
   - Revenue growth (>15% = secular tailwind)
5. **Pull performance** â€” 1Y/2Y returns via Polygon API
6. **Rank into tiers** â€” Tier 1 (own now), Tier 2 (accumulate), Tier 3 (watch)
7. **Identify catalysts + risks** per subsector
